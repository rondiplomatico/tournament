Das model-Package stellt Entitätsklassen der Anwendung zur Verfügung. Es ist  von anderen Packages unabhängig, außer der Klasse Tournament, welche eine  Verbindung zum planning-Modul hat. In diesem Package ist ein weiteres Package enum  enthalten. Dieses enthält Aufzählungsdatentypen, welche in Entitätsklassen verwendet  werden.
