Das control-Package enthält einen Teil die Steuerungsschicht der Anwendung und keine weitere Packages.  Die Klassen in diesem Package arbeiten direkt mit dem Datenmodell,  d.h. sie erstellen, aktualisieren, verändern und entfernen Daten.  Außerdem stellen sie Methoden nach außen bereit, die ermöglichen, das Datenmodell  anzusprechen.
